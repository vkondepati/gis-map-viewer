## üîç Quick Reference

### File Formats

**Vector Data**:

- Shapefile (.shp, .shx, .dbf)
- GeoJSON (.geojson, .json)
- CSV with coordinates
- KML/KMZ
- GeoPackage (.gpkg)

See [SUPPORTED_FORMATS.md](SUPPORTED_FORMATS.md)

### Data Warehouses

- DuckDB (local)
- Snowflake (cloud)
- Databricks (cloud)
- Apache Iceberg (open)

See [DATA_WAREHOUSES.md](DATA_WAREHOUSES.md)

### Keyboard Shortcuts

| Action   | Shortcut   |
| -------- | ---------- |
| Zoom In  | +          |
| Zoom Out | -          |
| Pan      | Arrow keys |
| Identify | I          |
| Select   | S          |
| Measure  | M          |
| Undo     | Ctrl+Z     |
| Redo     | Ctrl+Y     |
| Save     | Ctrl+S     |
| Open     | Ctrl+O     |

See [USAGE.md](USAGE.md) for complete list.
