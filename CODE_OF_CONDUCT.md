# Code of Conduct

This project follows the Contributor Covenant v2.1 as a baseline for community behavior.

Be respectful, be constructive, and welcome diverse perspectives. Harassment, hateful language, and personal attacks are not acceptable.

## Our Standards

- Treat everyone with respect.
- Be considerate in your communications and code reviews.
- Give constructive, actionable feedback.
- Respect privacy and personal boundaries.

## Reporting

If you experience or witness unacceptable behavior, please report it to the maintainers by opening a private GitHub issue or contacting the repository maintainers via GitHub Discussions. The maintainers will respond and follow up confidentially.

## Enforcement

Project maintainers will review reported incidents and take appropriate action, which may include:

- Requesting changes or apologies
- Temporary or permanent bans from the project
- Reporting to hosting platforms when required
